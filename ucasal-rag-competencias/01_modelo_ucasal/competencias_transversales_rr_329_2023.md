# Competencias Transversales UCASAL

De acuerdo con las referencias incluidas en la fuente consolidada, las competencias transversales comunes a las carreras de UCASAL son:

1. Trabajo en equipo.
2. Comunicación asertiva.
3. Habilidades de investigación.
4. Uso de herramientas digitales.
5. Autorregulación del aprendizaje.
6. Actuación ética.

## Uso RAG

Estas competencias deben recuperarse como marco transversal, pero no deben reemplazar a las competencias específicas de cada carrera. El agente debe diferenciar entre:

- Competencias transversales institucionales.
- Competencias específicas de carrera.
- Resultados de aprendizaje de asignatura.
- Evidencias de evaluación.
