# Diccionario de verbos para competencias y resultados de aprendizaje

| Nivel | Verbos sugeridos |
|---|---|
| Comprensión | explica, describe, interpreta, reconoce, identifica |
| Aplicación | aplica, utiliza, resuelve, implementa, emplea |
| Análisis | analiza, distingue, compara, relaciona, diagnostica |
| Diseño | diseña, formula, planifica, elabora, proyecta |
| Evaluación | evalúa, fundamenta, valida, justifica, pondera |
| Desempeño profesional | interviene, gestiona, asesora, dirige, supervisa, coordina |

## Recomendación

Evitar formular competencias solo con verbos como “conocer”, “comprender” o “saber”. Estos verbos pueden utilizarse en contenidos o propósitos generales, pero no son suficientes para expresar desempeños profesionales complejos.
