# Checklist de validación de programas por competencias

## Identificación

- [ ] La carrera está correctamente identificada.
- [ ] El plan de estudios corresponde al documento vigente.
- [ ] La modalidad está correctamente declarada.
- [ ] La resolución utilizada está citada en los metadatos.

## Competencias

- [ ] Se utilizaron competencias específicas oficiales de la carrera.
- [ ] No se inventaron competencias institucionales.
- [ ] Se integraron competencias transversales cuando correspondía.
- [ ] Las competencias están redactadas como desempeños complejos.

## Alineación curricular

- [ ] Cada competencia tiene resultados de aprendizaje asociados.
- [ ] Cada resultado tiene contenidos vinculados.
- [ ] Cada actividad permite ejercitar el desempeño esperado.
- [ ] Cada evaluación recoge evidencias del desempeño.
- [ ] Los criterios de evaluación son claros y observables.

## Validación final

- [ ] El docente responsable revisó la propuesta.
- [ ] La DDDI o área correspondiente realizó revisión tecnopedagógica.
- [ ] Se documentaron cambios y fuentes utilizadas.
